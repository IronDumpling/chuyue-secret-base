# Over-Clock Survivor

A third-person 3D survival game inspired by "Vampire Survivor" and "Backpack Heroes".

Player controls a robot whose danmaku attack is controlled by motherboard. Player needs to organize a tetris backpack to build up the motherboard in order to survive.
